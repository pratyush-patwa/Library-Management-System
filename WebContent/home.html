Terms and Conditions:

  1. Only Registered students can borrow a book from this Library Management System.
  2. Only one book is issued to a student at a time. If any student wants to borrow another book then he/she must return the previous books.
  3. A book will not be issued for more than Two weeks. If any student wants to have the book for more than two weeks, then he/she can borrow the same book again.
  4. Failure in returning the book on returning date will cost the student 10 rupee fine per day.
